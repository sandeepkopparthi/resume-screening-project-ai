## Sprint 1

### Problem

Need to process resumes in PDF format.

### Solution

Implemented a dedicated PDF parser to convert PDFs into plain text before sending them to the LLM.

### Benefit

Decoupled document handling from AI reasoning, making the system extensible to DOCX and TXT files.
